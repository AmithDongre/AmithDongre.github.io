## 🚀 Projects

### 🔐 CipherQuest: The IT Odyssey ⭐ Featured
A full-stack browser-based quiz game built and deployed for **Techotsav**, the inter-college IT fest at Alva's College. Teams navigate four themed worlds — Binary Jungle, Network Nebula, Code Citadel, and AI Realm — answering questions to unlock the next world.

- 90-question bank across 4 worlds and 3 difficulty modes
- Firebase real-time leaderboard syncing scores live across all teams
- Mathematically equal max score (315 pts) across all modes in every hint scenario
- Wrong-answer cooldown mechanic, hint deductions, boss questions, and a live host dashboard
- Built entirely through structured prompt engineering with Claude

**Tech Stack:** JavaScript · Firebase · HTML · CSS · Vercel · Prompt Engineering

🔗 [Live Demo](https://cipherquest-game.vercel.app) · [GitHub](https://github.com/AmithDongre/cipherquest-game)
