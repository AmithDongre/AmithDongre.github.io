Featured Project

CipherQuest: The IT Odyssey

Problem: Design a fair, competitive quiz game for a live inter-college IT fest where multiple teams play simultaneously across different difficulty modes.

Solution: Built a full-stack browser-based quiz game with a 90-question bank across four themed worlds — Binary Jungle, Network Nebula, Code Citadel, and AI Realm. Features a Firebase real-time leaderboard, three difficulty modes with mathematically equal max scores (315 pts), hint deductions, wrong-answer cooldowns, boss questions, and a live host dashboard. Deployed and used at Techotsav, Alva's College.

Built entirely through structured prompt engineering with Claude.
